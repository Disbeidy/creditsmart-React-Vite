## Descripción del proyecto
CreditSmart es una aplicación web desarrollada en **React** que permite a los usuarios:
- Explorar diferentes productos crediticios.
- Simular cuotas mensuales estimadas según monto, plazo e interés.
- Filtrar créditos por nombre y rango de monto.
- Completar un formulario de solicitud de crédito con datos personales, laborales y financieros.

El objetivo es brindar una plataforma confiable y fácil de usar para comparar y solicitar créditos en línea.

---

## Tecnologías utilizadas
- **React** (Frontend principal)
- **React Router DOM** (Navegación entre páginas)
- **Vite** (Herramienta de desarrollo rápida)
- **CSS personalizado** (Estilos y diseño responsivo)

## Instrucciones de instalación

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/Disbeidy/creditsmart-React-Vite.git
   cd creditsmart

2. Instalar dependencias:
   npm install

3. Ejecutar el servidor de desarrollo:
   npm run dev

4. Abrir en el navegador:
   http://localhost:5173/
